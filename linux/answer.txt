Вопрос №1
Вывести общее количество файлов из каталога/home/user1/
-type f -  ищет обычные файлы
wc -l - считает количество строк
Ответ:
find /home/user1/ -type f | wc -l

Вопрос №2
Допустим в каталоге больше 20 файлов.
Надо вывести названия файлов с 1 по 5, осортированных по времени создания начиная с новых.
Надо вывести названия файлов с 5 по 10, осортированных по времени создания начиная с новых.
Ответ:
Файлы с 1 по 5 по времени создания (новые первые)
ls -lt /home/user1/ | head -n 6 | tail -n 5
Файлы с 5 по 10 по времени создания
ls -lt /home/user1/ | head -n 11 | tail -n 6

Вопрос №3
Есть такой конфигурационный текстовый файл /home/user1/configuration_params.conf с такими данными: название - путь
WEBSERVER /usr/bin/nginx
NTP /usr/bin/ntpd
POSTFIX /usr/bin/postfix
Надо написать команду линукс, который выведет путь к NTP
Ответ:
 Путь к NTP из config файла
grep "^NTP" /home/user1/configuration_params.conf | awk '{ print $2 }'
